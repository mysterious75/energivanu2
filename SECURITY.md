# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ Active support  |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it privately:

- **Email:** support@voraprotocol.com
- **Do not** open a public GitHub issue

We will acknowledge receipt within 48 hours and provide a timeline for a fix. Security issues are prioritized above all other work.

## Scope

- API endpoints and authentication
- Data processing pipelines
- Network communications (OpenADR, SCED)
- Model checkpoint integrity

## Disclosure Policy

We follow coordinated disclosure:
1. Reporter notifies us privately
2. We develop and test a fix
3. We release the fix and notify the reporter
4. Public disclosure after 90 days or by mutual agreement
